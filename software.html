Software

My current development projects are on GitHub: @cmaumet. These include:

  • The NeuroImaging Data Model (NIDM): a set of specifications describing neuroimaging experiments.
  • nidmresults: a Python library to read and write NIDM-Results documents
  • nidmfsl: a Python library to export FEAT results from FSL into a NIDM-Results pack
  • SnPM: a Matlab library for non-parametric analyses in neuroimaging.


Previously, I developed:
  • RMagpie (BioConductor): a classification tool for microarray data.
